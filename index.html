The prompt

Please handle all the technical steps yourself and just ask me plain questions when you need answers. I would rather not run commands or edit files myself.

I want you to build me a personal AI assistant using these two GitHub repos:

1. Base framework: https://github.com/airtight-design/atd-bob-builder-kit
2. Memory add-on: https://github.com/jozuna19/assistant-starter-kit

Please do the whole setup for me, in this order:

STEP 1 - Read both repos so you understand the system. Clone or read the Bob Builder Kit and the Brain Vault Add-On yourself.

STEP 2 - Build my assistant using the Bob Builder Kit's full three-phase process, in order: first 01-discovery.md (interview me), then 02-architect.md (design my system from those answers), then 03-assembler.md (generate all the files). Walk me through each phase, asking a few simple questions at a time. I'll answer in plain language. Then generate all of my assistant's files (CLAUDE.md, SOUL.md, USER.md, AGENTS.md, SCHEMAS.md, MEMORY.md, TOOLS.md) for me.

STEP 3 - Install the Brain Vault Add-On for me. Copy the brain folder into my workspace, rename the template files, run its setup.sh so the memory hook is wired, and add the brain line to my CLAUDE.md. Do all of this yourself.

STEP 4 - Tell me when it's done and have me restart so the memory turns on. Then show me 2 or 3 simple things I can say to test it.

Important:
- Ask me to name my assistant and name my framework first (the default framework name is "the Matrix" if I don't have one).
- Do every technical step for me. I only answer questions.
- Never put any passwords or keys in files. Leave those marked [configure].

How it works

  1. 1

    Open Cowork

    Start a new Cowork session. That becomes your assistant's home.

  2. 2

    Paste the prompt

    Hit the copy button above, paste it into Cowork, and press enter.

  3. 3

    Answer the questions

    Claude interviews you in plain language. Pick a name for your assistant and answer honestly. When it asks to run a step, say yes.

  4. 4

    Restart and use it

    Claude finishes the setup and tells you to restart. Your assistant is now live and remembers you across sessions.

Takes about 30 minutes. The first week feels rough, correct it as it learns, and it gets sharper fast.

Never put passwords or keys in any file.

Stuck? Email bob@airtightdesign.com, the team that built the framework.
